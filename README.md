##  ■ はじめに
１つの区切りでしか報酬が得られない努力はたくさんあると思います。<br>
資格勉強も、年収UPまでの期間も、1日ずつ、１秒ずつ自分の努力は積み重なっているはずなのに、区切りの時期にしか評価が実感できない。<br>
そこで、１秒ずつの価値が可視化されればモチベーションも保ちやすいのではないかと思いこのアプリを開発しました。<br>

本リポジトリは、タイマーで努力時間を計測し、現在の努力の価値を金額として可視化できる、シンプルなアプリです。<br>

## ■ アプリの使い方
- まずは目標を設定。目標としている金額と、目標達成するまでに費やしたい時間を設定
- タイマースタートで計測が始まります。
- ストップを押すと、経過した時間で、今の自分の努力の価値が計算されます。
- 既にその目標に何時間か費やしていた場合、手入力で、設定した時間から努力を開始することも可能です。
