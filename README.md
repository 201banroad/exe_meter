##  ■ はじめに
本リポジトリは、タイマーで努力時間を計測し、現在の努力の価値を金額として、シンプルに可視化できるアプリです。<br>
![アプリ画面](readme_image/show.png)

## ■ アプリを作ったキッカケ
努力を毎日続けていると、１歩１歩進んではいることは分かっていても、たまに目標にどれくらい近づけているかの実感を失い、モチベーションも揺れる事がありました。<br>
そんな時に、どれくらい進めているかを「金額、努力の価値」として算出する事で、自分の現在地も大まかに把握でき、モチベーション維持に役立つと思いこのアプリを開発しました。

## ■ 特徴・コンセプト

## ■主な機能
- ユーザー登録・ユーザー削除・ログイン機能
- レスポンシブ対応
- タイマー機能（計測開始・計測停止・リセット機能）
- 現在の価値計算機能
- 目標金額・目標時間の設定機能
- 手動での経過時間変更機能
- 本日稼いだ額の表示機能
- １時間で稼げる額の換算機能
